## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Deployment

This app can be deployed as a static site to:
- Cloudflare Workers
- Raspberry Pi

Build the project with `npm run build` and deploy the `dist` folder.
